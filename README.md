## Install git   
Mac:  
```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew install git
```
Linux:  
```bash
sudo apt-get install git
```
Windows:    
Install git-bash      
https://gitforwindows.org/    
http://www.techoism.com/how-to-install-git-bash-on-windows/
